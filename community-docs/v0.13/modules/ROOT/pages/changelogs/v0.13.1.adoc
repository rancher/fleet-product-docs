== What's Changed

=== Auth fixes
* Add InsecureTLS and BasicHTTP options to Helm secret (#3872) by https://github.com/0xavi0[@0xavi0] in https://github.com/rancher/fleet/pull/3948
* Fix auth issues when downloading Helm charts from OCI registries by https://github.com/0xavi0[@0xavi0] in https://github.com/rancher/fleet/pull/3947

=== HelmOps
* Fix Helm access secret cloning (#3983) by https://github.com/0xavi0[@0xavi0] in https://github.com/rancher/fleet/pull/3988
* Support semver constraints with OCI polling by https://github.com/weyfonk[@weyfonk] in https://github.com/rancher/fleet/pull/3964
* Fix support for Helm repositories using `v` prefix in chart versions by https://github.com/0xavi0[@0xavi0] in https://github.com/rancher/fleet/pull/3959

=== Resources and statuses
* Prevent deployment errors caused by Content resource hash mismatches (#3917) by https://github.com/0xavi0[@0xavi0] in https://github.com/rancher/fleet/pull/3922
* Fix resources readiness detection in GitRepo status by @p-se in https://github.com/rancher/fleet/pull/3952
* Harden values files exclusion from bundle resources by https://github.com/weyfonk[@weyfonk] in https://github.com/rancher/fleet/pull/3956

=== Deployments
* Support graceful agent shutdown by https://github.com/weyfonk[@weyfonk] in https://github.com/rancher/fleet/pull/3927

=== Dependencies
* Update dependency go to v1.24.6 by https://github.com/thardeck[@thardeck] in https://github.com/rancher/fleet/pull/3971
* chore(deps): update gomod-k8s-dependencies to v0.33.4 (release/v0.13) by @renovate-rancher[bot] in https://github.com/rancher/fleet/pull/3998

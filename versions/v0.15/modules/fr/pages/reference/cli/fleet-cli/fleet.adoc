== VOIR ÉGALEMENT
=== Commandes principales

* xref:/reference/cli/fleet-cli/fleet_apply.adoc[fleet apply]     - Créer des bundles à partir de répertoires, et les exporter ou les appliquer sur un cluster
* xref:/reference/cli/fleet-cli/cleanup/fleet_cleanup.adoc[fleet cleanup]     - Nettoyer les enregistrements de cluster obsolètes
* xref:/reference/cli/fleet-cli/fleet_deploy.adoc[fleet deploy]     - Déployer une ressource de bundledeployment/contenu sur un cluster, en créant une release Helm. Cela ne déploiera pas les ressources de bundledeployment/contenu directement sur le cluster.
* xref:/reference/cli/fleet-cli/fleet_gitcloner.adoc[fleet gitcloner]     - Clone un dépôt git
* xref:/reference/cli/fleet-cli/fleet_target.adoc[fleet target]     - Afficher les cibles disponibles pour un bundle

=== Dépannage et diagnostics

* xref:/reference/cli/fleet-cli/fleet_monitor.adoc[fleet monitor]     - Surveiller les configurations de Fleet pour détecter des problèmes
* xref:/reference/cli/fleet-cli/fleet_analyze.adoc[fleet analyze]     - Analyser les instantanés créés par `fleet monitor`
* xref:/reference/cli/fleet-cli/fleet_dump.adoc[fleet dump]     - Exporter les données du cluster dans une archive

= gerenciamento de Fleet agent do controlador de frota
:revdate: 2025-07-23
:page-revdate: {revdate}

Configurar gerenciamento de Fleet agent

----
fleet-controller agentmanagement [flags]
----

== Opções

[source,text]
----
      --debug               Turn on debug logging
      --debug-level int     If debugging is enabled, set klog -v=X
      --disable-bootstrap   disable local cluster components
  -h, --help                help for agentmanagement
      --kubeconfig string   kubeconfig file
      --namespace string    namespace to watch
----

== Opções herdadas de comandos pai

[source,text]
----
      --disable-metrics   disable metrics
      --shard-id string   only manage resources labeled with a specific shard ID
----

== CONSULTE TAMBÉM

* xref:reference/cli/fleet-controller/fleet-controller.adoc[controlador de frota]     - Gerenciar o controlador de frota que se comunica com o Fleet agent

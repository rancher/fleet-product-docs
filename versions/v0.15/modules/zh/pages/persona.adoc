= 关键角色及其职责
:revdate: 2025-07-23
:page-revdate: {revdate}

[cols="1,2,2", options="header"]
|===
|角色 |功能 |主要优点

|平台工程师
a|* 在集群中设置和管理{product_name}安装。
* 定义并执行多集群应用程序和策略部署。
* 管理GitRepo和`fleet.yaml`配置。
* 执行集群范围内的更新和回滚。
a|* 在集群之间实现集中控制。
* 提高标准化和合规性。

|应用程序开发人员
a|* 在Git中构建应用程序Helm图表或清单。
* 在`fleet.yaml`中定义部署目标和环境。
* 通过UI或CLI监控部署状态。
a|* 基于GitOps的工作流程。
* 在各个环境中实现更快、自动化和一致的部署。

|安全管理员
a|* 定义并执行以代码形式的安全策略。
* 部署安全工具和配置。
* 审计和管理集群的安全态势。
a|* 降低风险并实现集中合规管理。
* 自动化、一致的安全配置。
|===
